# MARAG-Adaptive-Multilingual-RAG-for-Low-Resource-Languages
Official implementation of "Multilingual Adaptive RAG for Localized Knowledge" (ACM TOIS submission). Adaptive indexing framework for Swahili, Bengali, Amharic &amp; Spanish with 22.7% average improvement over baselines. Features language-aware routing, cross-lingual KG integration, and the MuLR benchmark.
